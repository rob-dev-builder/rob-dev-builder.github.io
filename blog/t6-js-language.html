JavaScript

Describe all the use cases for the following:

Parentheses ()

Can be used to execute a function. For example:

var result = doSomething()

Can also used to surround conditional statements like so:

if (count > 1) { CallFunction }

Parentheses are also used to control the order of operations:

(5 * 2(4 +2))

Brackets [ ]

Brackets can be used when you want to access the properties of an object. For example say you were describing an object called 'cat'

var cat = {name:'fluffy',colour:'grey'};

To get the colour of cat you can type:

cat['colour']

Brackets can also be used when accessing the properties of an array:

myArray[2]

Braces { }

Braces are used when you are describing an object in your code. You can see this in the cat example:

var cat = {name:'fluffy',colour:'grey'};

Braces are also used when you need to seperate parts of code. For example you would need to enclose some of your code with braces when defining a function:

var testFunction = function() {console.log("Some text")}

Single quotes ''

You can use single quotes when dealing with string variables. For example if you wanted to say 'Hello' you can write:

console.log('Hello');

Double quotes ""

Double quotes can also be used when dealing with string variables. It is useful to combine single and double quotes in situations like the following:

console.log("He said 'Hello' to her");

Double quotes must always be used when referring to an element in JSON object.
