Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

I think the best analogy is to imagine a house. The HTML is the structure of the house. Different rooms are used for different things like the elements of HTML. Each room is styled in a different way using different colours. This is similar to how CSS is used to style different elemetns in a HTML document.
The DOM (Document Object Model) is the logical structure of a webpage. Elements are arranged in a tree like structure with the html tag at the top and all other elements 'branching' down.

What is meant by boxifying design?

Boxifying design is taking a design document for a website and breaking the content down into box shapes. This helps to work out what tags might be used to desicribe the doucment in HTML.

What is the box model?

All elements in HTML are essentially boxes. The box model is used in CSS to describe the design and layout of an element. Since each element is essentially a box, every element has the following attributes in the box model:

  • Margin
  • border
  • Padding
  • Content
