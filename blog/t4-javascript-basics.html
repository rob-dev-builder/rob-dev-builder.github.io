How does JavaScript compare to HTML and CSS?

JavaScript allows you to add interactivity to your website. Where HTML provides the structure and CSS provides the styling. Javascript allows you to create animations, manipulate data, check for user input and more. Javascript is what helps make a static website dynamic.

Explain the difference between accessing data from arrays and objects.

When accessing data from an array you would use bracket notation '[]' and an index number to represent that item in the array.For example fruit[0] would return the first item in the 'fruits' array. To access data in an object you can use a 'key' value that represents the data you want. For example with 'bob.age' the key value 'age' in the bob object would return a number representing bobs age.

Explain what functions are and why they are useful.

Functions are blocks of code that can be called upon when needed. They are very useful because they help to reduce the amount of code you have to write. For example instead of writing out the same line of code many times (which would still work) You can put that same code into function and just call that function as many times as you need. This also helps to keep your code tidy and easy to maintain.
