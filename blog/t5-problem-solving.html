FizzBuzz Exercise Problem Solving:

During Sprint five of EDA training we had to do an exercise where we created a 'FizzBuzz' program using JavaScript. The FizzBuzz program tells the user which numbers (between zero and one hundred) are divisible by three, five and which numbers are divisible by both three and five.

When I first read the problem I was a bit nervous because I did not know where to begin to solve this problem. The first step to solve this problem was to use pseudocode. This is essentially writing out the steps of the program in simple english. By doing this it helped me to understand how to structure the flow of the program with out getting bogged down in the syntax of Javascript.

Once I had a good understanding of how the whole program would work with pseudocode, I started looking at how each function would actually work in JavaScript. Since I am still learning to code I didn't know how to write the syntax to get certain things done. Therefore, I opened up the Console in DevTools and started playing around with bits of code. The great thing about playing in the console is that I could quickly test ideas and see the result. For example to see to test if a number is divisible by three I just went into the console and typed: '10 % 4' Answer: 2

This confirmed for me how I could go about testing numbers in my program. Every time I got stuck or was not sure about how to code a bit of logic, I just went into the console and tried something out until I found a solution.

Sometimes I found myself spending a lot of time in the console and not really getting anywhere. In those situations I tried googling for an answer. For example, In this program I needed to use something called 'conditional operators'. This means that if my program came met certain conditions then it needed to do something. In my program I needed to test if a number was divisible by three and by five. to find out how to do this I went to google and found out about the '&&' operator. This allowed me to combine two conditions and test if they were both true at the same time.

Often I would get error messages when I tested my program. One error message that I kept getting was that a variable was 'undefined'. This meant that the program was not getting a piece of data that it needed to execute some code. When I got this error message I carefully scanned my code and looked at all my variable names. I would use a troubleshooting technique where you 'console.log' the variable. I often noticed that I would spell variable names incorrectly and this is what caused that problem.

During this exercise I never felt so blocked that I needed to use the rubber ducky method or ask my peers for help. Note: The rubber ducky method is a technique where you talk through the steps of your code to a rubber ducky (or some other object). However, I suspect there will be many times in the future when I will get stuck and will need to use these techniques.

Overall I felt really good about solving creating the program. I felt like I had the skills to solve issues as they came up. At first I was a bit nervous as I did not know how to start. However, as I started writing down code I became more relaxed and focussed. When I completed the program, I looked back at the code and reflected on how completed the exercise. I felt that I did a good job at tackling the problem in a logical way. I also liked the way I focussed on one small step at a time.

Describe what these functions do in your own words:

.map()

This function allows you to call a function on each element on an array. For example:

var arrayOfNames = ["Jim","Tim","Jane","Jess"];

var greeting = function(name){console.log("Hi " + name)}; arrayOfNames.map(greeting);

result: Hi Jim Hi Tim Hi Jane Hi Jess

.filter()

This function allows you to run each element in your array through a filter function. The filter function can be another function that you have created to sort through the items in the array.

For example: var ages = [32, 33, 16, 40];

function checkAdult(age) {return age >= 18;}; ages.filter(checkAdult);

result: [32, 33, 40]

.reduce()

The reduce function allows you to reduce an array down to a single value. The function will allow you to perform a function on each element on the array. For example, in an array of numbers you can reduce it down to a single total like so:

var testArray = [1,5]

testArray.reduce(function(total,num){return total + num})

result: 6
