Udacity Reflection

What is grid based design

Grid based design is breaking down your website layout into rows and columns. Using a gird system helps to control the layout of content on the page. This can be especially useful when making pages responsive.

Whats all the hype about responsive webpages?

More and more people are using browsers on mobile devices and such as tablets and smart phones. Resposinve webpages ensure that content is always displyed in a way that is supported on these smaller screens.

What is semantic structure?

Samantic tags are used to convey information about the content they contain. For example the tag for button is used to desribe a button element. This is different to structural tags such as div or span as they do not convey any meaning.
